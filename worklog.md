# npm
## 初期化
```
npm init -y
```

## npmのヘルプ
```
npm help i
```

## devDependenciesに追記してインスコ
```
npm i -D <package_name>
```

## オプションなしならデフォルトでdependenciesに追記
```
npm i <package_name>
```
